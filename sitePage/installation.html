Bioplentia Installation Guide

    
Bedrock Edition

• Open the File Explorer once you’ve downloaded an Bioplentia .mcaddon file;
• Go to your Downloads folder;
• Find the Bioplentia .mcaddon file and click it to import it to Minecraft;
• If you installed an Bioplentia, don’t forget to apply the Bioplentia resource pack and/or behavior pack when creating a new world or editing an existing world. And turn on Experimental Gameplay.

Any problems you have while importing Bioplentia are solely related to your device.
