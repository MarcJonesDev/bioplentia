What is Bioplentia?

Bioplentia is a Minecraft addon that aims to improve the look of vanilla biomes. Bioplentia greatly changes the look of vanilla biomes, and also adds new ones. Along with new biomes, you will find new blocks, new mobs, various structures etc.

Bioplentia is available for both editions of Minecraft: Bedrock and Java (experimental).
